# Assets Context

Static assets used by the add-in runtime, installers, and landing page.

## Contents

- App icons: `icon-16.png`, `icon-32.png`, `icon-64.png`, `icon-80.png`, `icon-128.png`.
- Ribbon icons in `assets/icons/` following the command-name convention.
- `draft-sticker.png` for the draft-marking slide action.
- `logo-filled.png` and related imagery used by the landing page and branding surfaces.

## Ribbon Icon Rules

- Most command icons are generated by `scripts/generate-icons.js`.
- Filenames follow `<actionName>_<size>.png`.
- The normal icon sizes are `16`, `32`, and `80`.

## Editing Guidance

- Prefer regenerating icon sets when the command list changes instead of hand-editing many PNGs.
- Keep `manifest.xml` `resid` references aligned with filenames.
- Delete icon files only when the corresponding command has been removed from the ribbon/taskpane and the manifest has been updated too.
