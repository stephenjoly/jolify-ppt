Quick Start

Ribbon First Task Pane Optional PowerPoint for Mac

After installation, PowerPoint shows a Jolify tab in the ribbon. The fastest way to get oriented is to use the first ribbon group:

  • Open Task Pane opens the side panel with Jolify’s broader toolset and theme color palette.
  • About Jolify opens this guide in your browser.

Most everyday work can stay in the ribbon. The task pane is best for deeper utilities like style clipboards, text presets, image deck creation, cleanup, and exports.

Selection Rules

Many Jolify tools follow one of a few selection models. Once you know these, the whole add-in becomes much easier to use.

Pattern How to Use It Examples
Reference shape Select the source shape first, then the target shapes. Jolify uses the first selected shape as the reference. Match Width, Match Height, Match Fill, Match Style, edge matching
Clipboard action Run a Copy action once, then run the matching Paste action on a new selection. Copy Position, Paste Size, Copy Fill, Paste Style
Multi-shape layout Select two or more shapes, then align, distribute, stack, or group them. Align Left, Distribute H, Stack Vertically, compound group actions
Exactly two objects Select exactly two shapes or text boxes. Swap Positions, Swap X, Swap Y, Swap Text
Slide selection Select slides in the left thumbnail pane, then run the slide-level command. Move to Unused, Save Selected Deck

Important: “Match” and “Copy/Paste” are different ideas in Jolify. Match applies directly from the first selected shape. Copy/Paste stores a setting for later use.

Task Pane Guide

The task pane exposes Jolify’s broader toolset. It mirrors many ribbon actions, but also includes extra clipboard actions, shape insertion tools, cleanup, export, and the deck color palette.

Insert

You can insert text boxes, rectangles, rounded rectangles, arrows, ovals, lines, elbow lines, symbols, Harvey balls, center stickers, and shape grids. Jolify inserts these centered on the current slide. Text-capable insertions use Jolify’s default text-box formatting.

Outline + Style

Use the Outline tools to cycle outline weight and dash presets or normalize selected lines to horizontal or vertical. Use Style Clipboard when you want a saved fill, outline, or full style you can paste later. Use Apply / Clear when you want to take the first selected shape and push its style directly onto the rest without storing anything.

Align

The task pane includes the core align and distribute tools plus resize-aware distribution and stack tools. Distribute with resize equalizes spacing and size together. Stack lays shapes out in one direction using Jolify’s fixed default gap.

Size

Use Match Size to copy dimensions from the first selected shape, Size Clipboard to save dimensions for later, and Match Edges to stretch shapes to a common left, right, top, or bottom boundary.

Arrange

The task pane adds the full order stack, group/ungroup, position clipboard actions, and compound align/distribute-plus-group actions.

Rotate

Rotate Right 90 and Rotate Left 90 rotate the current selection by exact quarter turns.

Swap

Swap Positions, Swap X, and Swap Y all require exactly two selected shapes.

Text

The text section holds Jolify-specific text-box actions:

  • Apply Default Textbox Format
  • Toggle Word Wrap
  • Merge Text Boxes
  • Split Text Box by Lines
  • Swap Text
  • Margin presets: None, Tight, Roomy
  • Auto-size presets: Off, Shrink text to fit, Resize shape to fit text

Other

The Geometry Clipboard lets you save and paste both position and size together. Utilities include:

  • Move to Unused
  • Weekday Range
  • Toggle Draft Sticker

Picture

Create Presentation from Pictures creates a new deck with one fitted image per slide. This is intended for local mode and image-driven workflows.

Share

The share section is Jolify’s publishing/export area:

  • Save Presentation as PPTX
  • Save Presentation as PDF
  • Save Selected Deck
  • Clean-up

Color Palette

The task pane’s right-hand color rail shows three columns:

  • Font applies text color
  • Outline applies line/outline color
  • Fill applies shape fill color

Jolify tries to read the current deck’s theme colors first. If that works, the badge shows Deck theme. If not, Jolify falls back to a generic palette.

PowerPoint behavior note: applying a color can drop focus from active text editing. If you were editing text inside a shape, you may need to re-enter the text cursor after changing color.

Exports & Cleanup

Save Presentation as PPTX / PDF

These save a copy of the full current presentation. In local mode, Jolify can use the local bridge for a more native save/open flow. In hosted mode, Jolify downloads the exported file.

Save Selected Deck

Select slides in the thumbnail pane, then use Save Selected Deck to create a new presentation containing only those slides.

Clean-up

Clean-up creates a new shareable copy of the deck. It does not modify the current open file. The cleanup dialog can remove:

  • slide comments
  • speaker notes
  • hidden slides
  • the Jolify unused-slides area
  • PowerPoint section metadata

Use this before sending client-ready decks or when you want a cleaner external version of a working file.

Local-mode only notes

Some workflows are richer in local mode because Jolify can hand files to the local bridge. The most obvious examples are picture-driven deck creation and some export/open flows.
