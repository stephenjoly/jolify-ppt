# Templafy Sample Icons Reference

This file maps the icons currently stored in `assets/sample-icons/` to the corresponding Templafy ProductivityPlus tools, summarizes how each tool works, and records Jolify’s current status against that tool.

Status legend:
- `Built` = available in Jolify today
- `Partial` = partly covered, indirect, or narrower than Templafy
- `Missing` = not currently available in Jolify
