Jolify

A PowerPoint ribbon add-in for shape precision

Stephen Joly PowerPoint for Mac MIT License

Abstract

Jolify is a native PowerPoint ribbon add-in that exposes shape positioning, alignment, formatting, and text utilities as one-click buttons. It was built to fill the gap in PowerPoint's built-in toolset for teams that work intensively with slide decks.

Once installed, a Jolify tab appears permanently in the ribbon. No task pane, no interruption — just the tools, exactly where you expect them.

Features

  • Position & Size Copy, paste, and swap shape positions and dimensions across any selection of shapes.
  • Match Propagate outline style, fill color, height, width, or all of the above from one shape to the rest.
  • Align Align shapes to each other or to the slide. Distribute three or more shapes evenly in any direction.
  • Text Split a multi-line text box into individual boxes — one per line — preserving position and size.
  • Branding Add or remove a DRAFT watermark on the slide master in a single click.

Installation

  1. Download the manifest.xml file below.
  2. Open PowerPoint → Insert → Add-ins → Upload My Add-in.
  3. Select the downloaded manifest file and confirm.
  4. A Jolify tab appears in the ribbon. Done.
